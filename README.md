📘 Student Management System
🚀 Project Description

This is a Full Stack Student Management System built using Spring Boot, MySQL, HTML, CSS, and JavaScript.
It allows users to manage student records using CRUD operations.

✨ Features
➕ Add new student
📋 View all students
✏️ Update student details
❌ Delete student
🔍 Search students (frontend)
⚠️ Form validation
📡 REST API integration
🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Spring Boot, Spring Data JPA
Database: MySQL

⚙️ How to Run
Clone the repository
Setup MySQL database
Run Spring Boot application
Open browser at http://localhost:8080
